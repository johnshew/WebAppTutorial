Bootstrap provides set of CSS styles that are widely used in modern web app development. These styles look good on most browsers running on a wide range of devices - from small phones to large displays.

Two important concepts in Bootstrap are predefined screen sizes and a 12 column layout grid.

The predefined screen sizes are designated xs, sm, md, and lg. For grid layout Bootstrap provides CSS classes based on these sizes. For example the CSS class col-sm-6 indicates that HTML element should cover 6 columns of the grid on sm devices and larger. If we have two HTML elements with col-sm-6 in a row then they will render as two columns on sm devices. If we have two HTML elements with col-lg-3 in a row then they will render as four columns on lg devices.
