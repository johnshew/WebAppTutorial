×Here we show how can seperate the "view" and the "code".
The view is just the HTML. And the code is now just in a function called a Controller.
We attach the controller function to the view with the ng-controller directive on the body element.
This means that there is now no code at all in the HTML.
When Angular sees the {{ variable }} syntax it binds to objects in $scope
Angular uses $scope as the "model" that glues together the view and the code.
