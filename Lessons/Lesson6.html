Here we show how can seperate the HTML view from the code.

To do this we moved the code that was in the HTML into a function called demoController. The view is now just the HTML we had before.

To connect the view with the code in the controller we attach both to a model. In Angular the model is a javaScript object called $scope. Notice the ng-controller directive on the body element. When it runs, the code the controller sets the property people. When Angular sees the {{ variable }} syntax it binds to properties in $scope. This use of $scope glues together the view and the code.

{{ person.name }}
Age: {{ person.age }}
