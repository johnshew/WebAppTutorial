This lesson shows using Bootstrap and Angular together.

Here we show the use of a Bootstrap panel to create a "Card" for each person. This kind of "card" have become an common user interface metaphor that we will use over the next few lessons.

A few more changes to note... We included the Bootstrap scripts so we can easily create an dismissable "alert" at the top of the page. And we moved all the script tags to end of the HTML body - in general this can help speed up rendering.
