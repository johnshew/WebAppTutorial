Here we introduce the other important javascript library we will be working with called Angular.

Note the use of the Angular directive ng-app on the html tag. This tells Angular that it should operate on this document. The directive ng-model directive binds the input box to the javaScript varible whatIsTyped. And finally {{ whatIsTyped}} displays this variable.

Try typing into the text box and notice how Angular does "binding" between the data in the text box and the HTML that follows. There is no "coding" required for this to work - it is very cool.
